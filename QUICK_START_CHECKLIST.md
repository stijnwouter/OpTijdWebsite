## 🐛 Troubleshooting

### Website laadt niet
- Check of alle bestanden correct zijn geüpload
- Check bestandsnamen (hoofdlettergevoelig!)
- Clear browser cache (Ctrl + Shift + R)

### Demo's werken niet
- Check browser console (F12)
- Check of `script.js` correct is geüpload
- Check of Confetti CDN laadt

### Formulier werkt niet
- Check FormSpree ID in beide bestanden
- Test of FormSpree account actief is
- Check spam folder voor test emails

### Screenshots laden niet
- Check of `/assets/` folder bestaat
- Check bestandsnamen exact
- Check image paths in HTML
